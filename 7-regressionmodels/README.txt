The purpose of this report is to concisely explore the relationship between a set of variables and miles per gallon(mpg) using the mtcars dataset. 
In particular, the report will attempt to address the following:  
- Is an automatic or manual transmission  better for MPG?  
- Quantify the mpg difference between automatic and manual transmissions.
